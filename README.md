# SUDOKU #
This game is comprised of 3 modes
* Classic Sudoku 
* Killer Sudoku
* Duidoku

## Getting Started
Clone the git repository and make sure the gson library is properly loaded as a dependency.

### Prerequisites
IntelliJ IDEA
Java 11
